Scientific Methodology

Multi-Source Data Fusion

Our AI system integrates:

  • Sentinel-3 OLCI chlorophyll data (proxy for algae blooms)
  • EMODnet microplastic measurements
  • Copernicus SST and current data
  • HYCOM hydrodynamic models

Data is normalized to 1km grid resolution with temporal alignment.

Neural Network Architecture

Custom transformer-based model with:

  • Spatial attention mechanisms
  • Multi-temporal input processing
  • Uncertainty quantification layers

Model trained on 5 years of global data with k-fold validation.

Microplastic-Algae Correlation

Leverages established scientific relationship:

MP = k × [Chl-a] × f(SST, currents, salinity)

Where k is region-specific coefficient derived from EMODnet validation data.

Continuous Validation

Automated validation pipeline:

  • Daily comparison with new EMODnet reports
  • Drift detection for model recalibration
  • Uncertainty propagation analysis
