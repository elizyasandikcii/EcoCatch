Scientifically Validated Methodology

PubMed

Sentinel-3 OLCI Imaging

Daily monitoring of chlorophyll-a concentrations using Sentinel-3 Ocean and Land Colour Instrument (OLCI). Validated by ESA with 92% accuracy.

Data processed through ESA's SNAP toolbox with atmospheric correction and sunglint removal.

PubMed

EMODnet Validation

Ground truth validation through EMODnet microplastic database with over 12,000 sampling points. Samples analyzed via FTIR spectroscopy following JPI Oceans protocols.

PubMed

Algae as Microplastic Proxy

Peer-reviewed correlation model: MP = k × [Chl-a] × f(SST, currents, region). Coefficient k derived from EMODnet validation data.

PubMed

Machine Learning Validation

Transformer-based architecture achieving 89% validation accuracy against ground truth data. Models incorporate ocean current data and sea surface temperature.
